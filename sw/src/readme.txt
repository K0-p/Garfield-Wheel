USB HID Garfield Device
authors: Gabriel Moore and Calvin Heischman
*note: doesn't compile on uVision v5, get v4 ya dingus

This project is designed to run on the TM4C123GH6PM. It turns the device into a generic HID USB gamepad device.
Functionality:
	3-axis variable input
	8 buttons
	Can play Garfield Kart
	
Prototyped housing can be found in hw/wheel_models/
